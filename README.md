## Django project to share feed api

### Preconditions:
- Python 3

### Clone the project
```
git clone https://github.com/winston053790/django-share-feed.git
```

### How to run project

#### Install dependencies
```
pip install -r requirements.txt
```

#### Run server
```
python manage.py runserver
```

#### Run test
```
python manage.py test
```
